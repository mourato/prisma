# keychain-security

Guide for secure credential storage using macOS Keychain in the Meeting Assistant project.

## Overview

This skill provides patterns for using KeychainManager, handling Keychain errors, and managing secure storage for API keys, tokens, and other sensitive data.

## When to Use

Use this skill when:
- Storing or retrieving API keys, tokens, or passwords
- Working with KeychainManager or KeychainProvider
- Handling Keychain errors
- Implementing secure credential rotation

## Key Topics

- KeychainManager usage patterns
- Error handling strategies
- Secure storage best practices
- Biometric authentication integration
